# Uttori Storage Provider - JSON File

Uttori storage provider using JSON files on disk.

# Config

```js
{
  content_dir: '',
  history_dir: '',
  data_dir: '',
  extension: 'json',
}
```

## Tests

To run the test suite, first install the dependencies, then run `npm test`:

```bash
$ npm install
$ DEBUG=Uttori* npm test
```

# Namesake
